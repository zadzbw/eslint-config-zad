# eslint-config-zad

[![npm version][npm-badge]][npm-url]
[![node version][node-version-badge]][npm-url]
[![MIT license][license-badge]][mit-url]

* one-line eslint config for React.js
* single quotes
* no semi
* support typescript

## Usage

### Install

```bash
yarn add eslint eslint-config-zad -D 
```

### Config `.eslintrc.js`

```javascript
module.exports = {
  extends: ['zad']
}
```
